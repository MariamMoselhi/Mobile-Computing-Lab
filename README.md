## Getting Started

To run any project locally, navigate to the project folder and use the standard Flutter commands:

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/Mobile-Computing-Lab.git

# 2. Navigate to a project
cd Mobile-Computing-Lab/todo-list-app

# 3. Install dependencies
flutter pub get

# 4. Run the app
flutter run
```

> **Prerequisites:** Flutter SDK ≥ 3.0, Dart ≥ 3.0, and an emulator or physical device.
